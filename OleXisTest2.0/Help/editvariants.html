Редактирование вариантов ответа

Общее описание окна

В зависимости от выбранного типа вопроса, окно может содержать различные компоненты:
  • Кнопка "Добавить вариант" служит для добавления дополнительного варианта ответа;
  • Кнопка "Удалить" позволяет удалить лишний вариант ответа. Располагается возле каждого варианта, если тип вопроса предусматриает добавление вариантов ответа;
  • Текстовое поле для ввода вариантов ответа. Располагается возле каждого варианта, если тип вопроса предусматриает добавление вариантов ответа. Тип вопроса "Установление соответствия" предусматривает наличие двух текстовых полей: для понятия и его соответствия;
  • Кнопки со стрелками, которые позваляют изменить порядок последовательности
  • Флажки "Правильный ответ", с помощью которых отмечается правильный ответ. В типе вопроса "Установление соответствия" правильные соответствия задаются для каждого варианта. В типе вопроса "Установление последовательности" порядок следования вариантов образует правильную последовательность.
