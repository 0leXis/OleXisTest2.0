Установка и настройка сервера

Установка MySql

Если MySql уже установлен, перейдите к разделу "Установка сервера".
  1. Запустите файл MySqlSetup.exe и следуйте инструкциям установщика. Внимание! Путь к папке с MySql не должен включать символы кириллицы!
  2. Чтобы запустить сервер, откройте файл MySqlServer.bat в папке установки, либо его ярлык на рабочем столе

Установка сервера

  1. Запустите файл ServerSetup.exe и следуйте инструкциям установщика
  2. Чтобы запустить сервер, откройте файл OleXisTestServer.exe в папке установки, либо его ярлык на рабочем столе
  3. Нажмите кнопку "Настройки сервера"
  4. Если вы установили MySql по инструкции выше, то все значения уже будут настроены. Если вы используете свою версию MySql, измените настройки в соответствии с вашей версией.
  5. Для запуска сервера нажмите кнопку "Запустить сервер". До нажатия на кнопку убедитесь, что MySql запущена и все настройки введены правильно.
